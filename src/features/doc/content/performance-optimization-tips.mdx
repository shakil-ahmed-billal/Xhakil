Performance is a key part of user experience. A slow site can drive users away, no matter how good the design is.

## 3 Quick Tips for Better Performance:

- **Image Optimization**: Always use modern formats like WebP or AVIF.
- **Code Splitting**: Only load the JavaScript your users need for the current page.
- **Caching**: Leverage pre-rendering and edge caching for instant load times.

Speed is a feature!
